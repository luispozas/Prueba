JustGetMyData

Información del proyecto

  • Categoria: Web
  • Project date: 01 Enero, 2021
  • Project URL: www.justgetmydata.com

Numerosas empresas usan una técnica llamada "dark pattern" para dificultar la obtención de datos personales en sus servicios web. JustGetMyData busca ser un directorio donde centralizar los enlaces para obtener fácilmente sus datos personales.

Este proyecto ha tenido una enorme repercusión, llegando a numerosos rincones del mundo.
